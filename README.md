## 9-5
1. **线段树**
   - 线段树说明
     - 用二叉树存储区间，每一个节点表示一个区间范围
     - 满足条件: 区间[L, R] 可以由[L, M] 和 [M + 1, R]求出
     - 满足问题: 区间求和，区间最大值最小值
     - 不满足问题: 区间的众数，区间连续最长问题
     - 数据结构形式: tree{left: 左范围，right: 右范围，value: 值，lazy: 更新值}，eg. tree{left = 1, right = 2, value = 1, lazy = 1}，表示其范围为[1, 2], 当前值为1，lazy值为1(需要向其子区间传递相应值更新)
   - log2整数部分快速求法
